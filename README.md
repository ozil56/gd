# 掼蛋记分器 2.0

基于文件存储的掼蛋记分器，支持通过牌局 ID 分享和同步即时比分。

## 部署说明

1. php环境，并打开

extension=sqlite3

extension=pdo_sqlite

3. 默认文档设置为 `index.html`

## 使用说明

1. 第一个人创建新牌局后，点击左上角“复制链接”，将该链接发给同桌的其他牌友，他们打开该链接即可进入当前牌局。
2. 每个人打开当前牌局的人都可以进行记分、修改队名等操作。修改将自动同步（同步间隔2秒）
3. 误操作后，可点击下方“撤回上一步”。
4. 当前庄家会有一个皇冠图标标注提示。
5. A失守3次即回到2。
